QScript Commands

var NAME VALUE

 # Create a variable
var greeting "Hello, world!"

# Use the variable in a command
write console $greeting
Arguments NAME: The name of the variable; this will be used with interpolation (see below). Each variable name must be unique in a given script.
VALUE: The variable's value.
Description
 Variables allow scripts to store bits of information for reuse. All inputs for all commands are interpolated; each input will have the value of each variable "injected" into them. For each variable, every instance of "$" followed by that variable's name will be replaced with the variable's value.  For example, assume that a script has two variables, one named "greeting", with the value of "Hello", and "target", with the variable of "world". If an input of "$greeting $target!" is interpolated, it will result in that input becoming "Hello world!".


set VARIABLE VALUE

 # Create a variable
var greeting "Hello"

# Set the "greeting" variable to a new value
set greeting "Goodbye"
Arguments VARIABLE: The name of the variable to change.
VALUE: The variable's new value.
Description
 Changes a variable's value.


raw COMMAND ARG[...]

 # Join #channel using "test" as a key
raw join #channel "test"
Arguments COMMAND: The IRC command to issue.
ARG[..]: The argument(s) to pass with the command
Description
 Sends a raw command to the IRC server


write TARGET TEXT

 # Write to the server window
write server "Hello world!"

# Write to a channel window
write #mychannel "This is the #mychannel window"
Arguments TARGET: The name of the window to write to; server will cause text to be written to the server window, console will cause text to be written to the console, and the name of a channel or user will cause text to be written to the channel or user's window.
TEXT: The text to write.
Description
 Writes text to a Quirc window.  Note: this will only print text to the window.


sleep NUMBER

 # Wait for 10 seconds
sleep 10
Arguments NUMBER: The number of seconds to pause.
Description
 Pauses the desired number of seconds before executing the next command.


msleep NUMBER

 # Wait half a second, or 500ms
msleep 500
Arguments NUMBER: The number of milliseconds to pause.
Description
 Pauses the desired number of milliseconds before executing the next command


join CHANNEL [KEY]

 # Join the #quirc channel
join #quirc

# Join #example, locked with key "passwd"
join #example "passwd"
Arguments CHANNEL: The name of the channel to join.
KEY: Optional. The channel's key, if required.
Description
 Joins a channel.


part CHANNEL

 # Leave the #quirc channel
part #quirc
Arguments CHANNEL: The name of the channel to leave.
Description
 Leaves a channel.


nick NEW_NICKNAME

 # Change the current nickname to "bob"
nick "bob"
Arguments NEW_NICKNAME: The nickname to change to.
Description
 Changes your nickname. If the nickname is already in use, the new nickname with "_" appended to the end will be used.


msg TARGET TEXT

 # Send a greeting to everyone in #quirc
msg #quirc "Hello, everyone!"

# Send a private message to alice
msg alice "Hello, alice!
Arguments TARGET: The name of the channel or user to send the message to.
TEXT: The message to send.
Description
 Sends a private message.
